# 环境变量参考

配置来源是项目根目录 `.env`。不要把真实 `.env`、密钥或数据库密码提交到 Git。

## 基础配置

| 变量 | 默认值 | 说明 |
|---|---|---|
| `NODE_ENV` | `development` | `development`、`test`、`production` |
| `PORT` | `7319` | HTTP 监听端口 |
| `DATABASE_URL` | 本地 PostgreSQL 示例 | Prisma 数据库连接串 |
| `LOG_LEVEL` | `info` | Pino 日志级别，代码直接读取 |

## 飞书配置

| 变量 | 默认值/要求 | 说明 |
|---|---|---|
| `FEISHU_APP_ID` | 空 | 企业自建应用 App ID |
| `FEISHU_APP_SECRET` | 空 | 应用密钥 |
| `FEISHU_VERIFICATION_TOKEN` | 生产必填 | 回调 token 校验 |
| `FEISHU_ENCRYPT_KEY` | 生产必填 | 加密回调解密密钥 |
| `FEISHU_API_BASE_URL` | `https://open.feishu.cn` | 飞书 API 根地址 |
| `FEISHU_NOTIFY_RECEIVE_ID_TYPE` | `chat_id` | `open_id`、`user_id` 或 `chat_id` |
| `FEISHU_NOTIFY_RECEIVE_ID` | 空 | 固定财务用户或群；空值不发送固定通知 |
| `FEISHU_APPROVAL_DETAIL_URL_TEMPLATE` | 飞书审批 AppLink | 支持 `{instanceCode}` 占位符 |
| `APPLICANT_NAME_MAP` | 空 | ID 到姓名的临时映射，逗号分隔 |

## 附件下载安全

| 变量 | 默认值 | 说明 |
|---|---|---|
| `FEISHU_FILE_DOWNLOAD_ALLOWED_HOSTS` | 飞书/Lark 官方域名列表 | HTTPS 直链下载允许域名，支持 `*.example.com` |
| `FEISHU_FILE_DOWNLOAD_MAX_BYTES` | `20971520` | 直链附件最大 20 MiB |

## OCR 配置

| 变量 | 默认值 | 说明 |
|---|---|---|
| `OCR_PROVIDER` | `mock` | `mock`、`openai`、`ai-vision`、`openai-compatible` |
| `AI_VISION_API_KEY` | 回退到 `OPENAI_API_KEY` | AI OCR 密钥 |
| `AI_VISION_BASE_URL` | 回退到 `OPENAI_BASE_URL` | 模型服务地址 |
| `AI_VISION_MODEL` | 回退到 `OPENAI_VISION_MODEL` | 模型名称 |
| `AI_VISION_API_STYLE` | `responses` | `responses` 或 `chat_completions` |
| `OPENAI_API_KEY` | 空 | 兼容旧配置 |
| `OPENAI_BASE_URL` | `https://api.openai.com` | 兼容旧配置 |
| `OPENAI_VISION_MODEL` | `gpt-4o-mini` | 兼容旧配置 |

详见 [[06-OCR与附件处理]]。

## 审批字段映射

| 变量 | 默认候选名称 |
|---|---|
| `APPROVAL_AMOUNT_FIELD_NAMES` | `费用明细汇总,报销金额,金额,实付金额` |
| `APPROVAL_ATTACHMENT_FIELD_NAMES` | `付款凭证,支付截图,报销凭证,附件,图片/视频,图片` |
| `APPROVAL_INVOICE_FIELD_NAMES` | `发票,发票附件,电子发票,发票图片` |
| `APPROVAL_APPLICANT_FIELD_NAMES` | `申请人,报销人` |
| `APPROVAL_HANDLER_FIELD_NAMES` | `办理人,经办人,付款办理人` |

候选项用英文逗号分隔，匹配时忽略大小写并去除两端空格。

## 存储、查重和 Worker

| 变量 | 默认值 | 说明 |
|---|---|---|
| `LOCAL_STORAGE_DIR` | `./data/evidences` | 原件保存目录 |
| `SAVE_ORIGINAL_FILE` | `true` | APPROVED 阶段是否保存原件 |
| `SAVE_OCR_RAW_TEXT` | `false` | 是否保存完整 OCR 原文 |
| `PERCEPTUAL_HASH_DISTANCE_THRESHOLD` | `8` | 64 位感知哈希最大汉明距离 |
| `AUDIT_WORKER_POLL_MS` | `1000` | 审核和通知轮询间隔 |
| `AUDIT_JOB_LEASE_SECONDS` | `300` | 审核任务租约秒数 |
| `AUDIT_JOB_MAX_RETRIES` | `5` | 审核任务最大尝试次数 |

## 推荐模板

```env
NODE_ENV=production
PORT=7319
DATABASE_URL=postgresql://USER:PASSWORD@HOST:5432/feishu_approval

FEISHU_APP_ID=your_app_id
FEISHU_APP_SECRET=your_app_secret
FEISHU_VERIFICATION_TOKEN=your_verification_token
FEISHU_ENCRYPT_KEY=your_encrypt_key
FEISHU_NOTIFY_RECEIVE_ID_TYPE=chat_id
FEISHU_NOTIFY_RECEIVE_ID=your_chat_id

OCR_PROVIDER=openai-compatible
AI_VISION_API_KEY=your_key
AI_VISION_BASE_URL=https://your-provider.example/v1
AI_VISION_MODEL=your_vision_model
AI_VISION_API_STYLE=chat_completions

SAVE_ORIGINAL_FILE=true
SAVE_OCR_RAW_TEXT=false
```
